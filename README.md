## Coding Factory 9 AUEB Java Projects

A comprehensive collection of Java projects, exercises, and architectural implementations, demonstrating the progression from Structural Programming (POP) to Advanced Object-Oriented Programming (OOP) and Software Architecture.

## Repository Structure

This repository is divided into two major parts, reflecting the learning curve of modern Java Development.

---

### Part 1: Procedure-Oriented Programming (POP)
*Foundational concepts, algorithmic thinking, and basic syntax.*

| Chapter | Topic                                       | Description                                                                                           |
| :--- |:--------------------------------------------|:------------------------------------------------------------------------------------------------------|
| **Ch 01** | **Introduction to Java**                    | Setting up the environment, Hello World, basic syntax.                                                |
| **Ch 02** | **Data Types (Integer)**                    | Understanding `int`, memory, and arithmetic operations.                                               |
| **Ch 03** | **Boolean & Control Flow (Part A)**         | `boolean` type, `if/else` conditionals, and `while` loops.                                            |
| **Ch 04** | **Char & Control Flow (Part B)**            | `char` type (Unicode), `for` loops, and `switch` statements.                                          |
| **Ch 05** | **Floating Point & Structured Programming** | `float`/`double` precision and modular programming concepts.                                          |
| **Ch 06** | **Arrays**                                  | Complex data types: 1D and 2D arrays, algorithms, and traversing.                                     |
| **Ch 07** | **Strings**                                 | String manipulation, immutability, and standard library methods.                                      |
| **Ch 08** | **Exception Handling**                      | Introduction to error handling: `try-catch` blocks.                                                   |
| **Ch 09** | **File I/O**                                | Reading from and writing to files (Input/Output streams).                                             |
| **Ch 10** | **Review and Challenges**                   | Consolidating Part 1 concepts and small-scale applications combining structured programming concepts. |

---

### Part 2: Object-Oriented Programming (OOP) & Core Libraries
*Advanced concepts, design patterns, data structures, and enterprise architecture.*

| Chapter | Topic | Description                                                                  |
| :--- | :--- |:-----------------------------------------------------------------------------|
| **Ch 11** | **Introduction to OOP** | Classes, Objects, and the shift from procedural to object-oriented thinking. |
| **Ch 12** | **Public API** | Designing classes, encapsulation, and method exposure.                       |
| **Ch 13** | **Utility Classes & Singletons** | Static methods, Factory methods, and the Singleton pattern.                  |
| **Ch 14** | **Inheritance & Polymorphism** | `extends`, Overriding, `super`, and Custom Exceptions.                       |
| **Ch 15** | **Interfaces & Abstraction** | Abstract classes, Interface contracts, and decoupling.                       |
| **Ch 16** | **Advanced Class Features** | Nested/Anonymous classes, **Lambdas**, Method References, and Mixins.        |
| **Ch 17** | **Generics & Collections** | Java Collections Framework (Lists, Sets, Maps), Generics, and Enums.         |
| **Ch 18** | **Layered Architecture** | Full-scale application design (DTO, DAO, Service, Controller).               |
| **Ch 19** | **Regular Expressions** | String pattern matching, validation, and Regex API (`Pattern`, `Matcher`).   |

---
